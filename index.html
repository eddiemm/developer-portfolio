I am a front end developer with a passion for bringing web designs to life. I love debugging annoying CSS properties, exploring sites for microinteraction opportunities, or finding creative ways to display data to the user. It's kinda my thing.

Developer Philosophy

Every pixel has to be perfect

Every aspect of a web design is calculated. Leave it up to me to implement it to look exactly as it was intended across all browsers and devices using modern CSS techniques.

Every edge case needs to be considered

Efficiency is always important. I research and plan before writing code followed by thorough refactoring to ensure code is as optimized as possible.

Every user needs to be to thought of

Every user should have the opportunity to navigate, view, and use your website. Accessibility should not be sacrificed for design. Let me focus on implementing accessibility and testing. I can ensure your web product follows the standard Web Content Accessibility Guidelines and use available tools to test for accessibility issues.
